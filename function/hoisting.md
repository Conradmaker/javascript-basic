# Hoisting

호이스팅은 사전적 의미로 끌어올리는 것을 의미합니다.

그렇다면 무엇을 끌어올릴까요? 

* 변수 '선언'
* 함수 '선언'

즉, 선언을 끌어올리게 됩니다. 

호이스팅은 선언과 밀접한 관련이 있는데, 자바스크립트 엔진은 코드를 실행하기에 앞서 선언되 내용이 있는지 코드를 한번 쭉 살펴보고, 발견하면 위로 끌어올려 줍니다.
